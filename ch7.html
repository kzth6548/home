챕터 7. 자바스크립트 코어 객체와 배열

여기에서는 자바스크립트의 코어 객체들의 활용을 알아 볼 것입니다. 차례로 예시가 이어집니다.

7.3 배열과 Array

자바스크립트에서 Array 객체를 사용하여 배열을 만들 수 있다. 아래는 Array 객체로 배열을 만들어 이름을 저장 및 출력해 보았다.
<script>
var degrees = new Array(); // 빈 배열 생성
degrees[0] = "김재혁";
degrees[1] = "김금숙"; 
degrees[2] = "김승현";
degrees[3] = "김우현"; 
degrees[4] = "김주현"; 

document.write("결과 "+"<br>");
for(i=0; i<degrees.length; i++)
	{document.write("degrees[" + i + "] = " + degrees[i] + "<br>");}</script>

7.4 Date

Date 객체는 시간과 관련된 정보를 담고 있다. 아래의 예제는 방문한 시각에 따라 다른 그림이 출력되도록 하였다.

<canvas id="myCanvas" 
		style="background-color:aliceblue"
	 	width="300" height="300"></canvas>
<script>
	var canvas = document.getElementById("myCanvas");
	var context = canvas.getContext("2d");
	var current = new Date();
	var img = new Image();
	
	img.onload = function () {
		context.drawImage(img, 0, 0, canvas.width, canvas.height);
	}
	if(current.getSeconds() % 2 == 0)
	img.src = "media/android.jpg";
else
	img.src = "media/iphone.png";
</script>

7.5 String

String 객체를 사용하여 문자열을 저장하고 다룰 수 있다. 여기에서는 일반 문자열과 string 객체에 저장된 문자열을 eval 함수에 넣었을 때의 차이를 확인해 보았다. 

<script>
var s1 = '2 + 2';
var s2 = new String('2 + 2');
document.write(eval(s1) + "<br>");
document.write(eval(s2) + "<br>");
</script>

7.6 Math

Math 객체는 자바 스크립트 내에서 수학 계산을 위해 사용하는 객체이다. new Math()로 생성하진 않으며 Math.프로퍼티 또는 Math.메소드()를 통해 사용한다. 밑의 예제에서는 Mate 객체의 random 메소드를 사용하여 난수 10개를 만드어 보았다.

<script>
for(i=0; i<10; i++){
	var m = Math.random()*100;
	var n = Math.floor(m);
	document.write(n + " ")
}
</script>
