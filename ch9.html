9.4 마우스 핸들링

마우스 이벤트 객체의 프로퍼티와 이벤트 리스너를 활용할 수 있다. 여기 예제에서는 마우스 이벤트 리스너의 기본인 onmouseover / onmouseout / onmousedown / onmouseup을 사용한 예제이다.
 <p onmouseover="this.style.backgroundColor='red'"
   onmouseout="this.style.backgroundColor='white'"
   onmousedown="this.style.color='blue'"
   onmouseup="this.style.color='black'">
마우스를 올리면 글씨의 배경 색이 바뀌고 클릭하면 글씨 색이 바뀝니다. </p>

9.5 문서와 이미지 로딩, onload

onload는 문서나 이미지가 로딩 되었을 때를 포착하는 이벤트 리스너이다. 여기 예제에서는 이미지가 로딩되었을 때 그의 관련된 정보를 출력하는 예제이다. 
 
<ch9.5>
<script>
function changeImage() {
	var sel = document.getElementById("sel");	
	var img = document.getElementById("myImg");
	img.onload = function () {
		 // 이미지 크기 출력
	 	var mySpan = document.getElementById("mySpan");
		mySpan.innerHTML = img.width + "x" + img.height;	
	}
	var index= sel.selectedIndex; // 선택된 옵션 인덱스
	img.src = sel.options[index].value; // <option>의 value 속성
	// 이미지 로딩 시작, 완료 후 onload 리스너 호출
}
</script>
<form>
<select id="sel" onchange="changeImage()">
	<option value="media/android.jpg">안드로이드
	<option value="media/iphone.png">아이폰
</select>
<span id="mySpan">이미지 크기</span>
</form>
<p><img id="myImg" src="media/android.jpg" alt="."></p>
</ch9.5>

9.6 폼과 이벤트 활용

앞서 배웠던 폼에서 발생하는 이벤트를 활용하여 문서의 다채로운 변화를 줄 수도 있다. 아래의 예제에서는 selete 객체에서 선택한 휴대폰을 출력한다.

<ch9.6>
<script>
function drawImage() {
	var sel = document.getElementById("phone");
	var img = document.getElementById("phoneimage");
	img.src = sel.options[sel.selectedIndex].value;
}
</script>
<form>
<select id="phone" onchange="drawImage()">
	<option value="media/android.jpg">안드로이드
	<option value="media/iphone.png" selected>아이폰
</select>
<img id="phoneimage" src="media/iphone.png" width="300" height="300" alt="">
</ch9.6>

이벤트와 이벤트 리스너의 활용에 대해 알아보았다. 이를 통해 동적인 웹 페이지 구현을 할 수 있을 것이다.
